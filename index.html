❓ HOW TO USE — UPPCL TOOL
🔐 LOGIN
Admin PIN0107Full access — ATR update, Sheet sync, Export
User PIN1234View, Search, Filter, Nearby — No Sheet sync
🔍 SEARCH TAB
  • Enter Account ID, Mobile Number or Serial Number
  • Full consumer details popup opens with all info
  • Click 📍 OPEN IN GOOGLE MAPS to view location
  • Click 🔗 COPY LINK to share consumer details link
⚙️ FILTER & EXPORT TAB
  • Filter by Substation → Feeder (feeder updates automatically)
  • Filter by Category and Load range (kW)
  • Summary cards show matched count by category
  • Export filtered results as XLSX or CSV
  • S.NO resets automatically when filter changes
📍 NEARBY TAB
  • Click 📡 USE MY GPS LOCATION — finds consumers near you
  • Click 🗺️ CLICK ON MAP — click any point on map to search
  • Set Radius (250m to 10km)
  • Filter by Substation / Feeder / Category
  • Results sorted by distance — export as XLSX/CSV
🔴 SM UNRECHARGE TAB
  • Shows all 1,234 Smart Meter consumers who have not recharged after disconnection with outstanding amounts
  • Click any Account ID to open full details popup
  • Inside popup: set ATR Status → PAID / LOCK / THEFT / OTHER
  • Click 💾 SAVE ATR STATUS → auto-syncs to Google Sheet
  • Popup also shows nearby disconnected consumers automatically
  • Filter by Substation / Feeder / LMV / ATR Status / Outstanding range
  • 📋 EXPORT ATR REPORT — download all ATR statuses as XLSX
  • Bottom section: 📍 Nearby Disconnected — GPS or map pick
☁️ GOOGLE SHEET SYNC (Admin only)
  • Click ☁️ SHEET SYNC SETTINGS in Disconnected tab
  • Paste your Apps Script Web App URL → click CONNECT
  • ATR changes auto-save to Google Sheet instantly
  • Click 🔄 SYNC NOW to pull latest data from sheet
  • All team members see same ATR data in real time
